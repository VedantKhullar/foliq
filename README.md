# Foliq

Foliq is a personal intelligence dashboard being built to organize and monitor information across different domains — from markets and macro trends to signals, events, and evolving narratives.

The project focuses on building a clean and scalable system that combines structured data, visual dashboards, and AI-assisted workflows to make complex information easier to track, explore, and understand over time.

## Tech Stack

* Angular
* TypeScript
* SCSS
* RxJS
* Node.js

Currently in active development.
